## Style Guide

You can extract documentation from any CSS file by using the `<CssDoc>` component:

```jsx
<CssDoc filePath="path/to/your/css/file.css" title="Optional Title" />
```

The component will automatically parse JSDoc-style comments from your CSS files and display them in a beautiful format.

## Supported Comment Formats

- `@param` - Document parameters or modifiers
- `@example` - Show usage examples
- `@since` - Version information
- `@deprecated` - Mark as deprecated
